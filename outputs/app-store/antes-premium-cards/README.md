# Antes App Store Screenshot Pack

Pack premium para os cards/screenshot carousel do App Store Connect.

## Conteúdo

- `final/`: PNGs finais em 1290 x 2796.
- `review/review-board.html`: página local para revisar todos os cards.
- `data/cards.json`: mensagens e estrutura dos cards.
- `app-store-description.md`: descritivo, subtítulo, texto promocional e keywords sugeridas.
- `polish-layers/`: camadas visuais geradas sem texto nem UI.

## Direção criativa

Textos e telas do produto são renderizados deterministicamente. A camada generativa é usada apenas como fundo premium, sem textos, logos, números, claims ou UI.
